# Number Printer

This is a simple Python program that prints numbers from 1 up to the number given by the user.  
The purpose of this project is to practice basic Python concepts such as loops, functions and input validation.

## What the program does
- Asks the user to enter a number
- Prints all numbers from 1 to the entered number
- Checks whether the input is a valid number
- Allows the user to exit by typing "q"
- Uses a function to keep the code organized

## How to run
1. Install Python on your computer
2. Download or clone this repository
3. Open a terminal in the project folder
4. Run the following command:
5   python number_printer.py
6. Enter a number when asked  
7. To exit the program, type `q`
